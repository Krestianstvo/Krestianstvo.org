  • Based on Open Qwaq version of the Open Croquet distributed computation architecture
  • Squeak Smalltalk programming language
  • Zero install - running anywhere without setup
  • NoSQL CouchDB database is used for services serialization, instead of platform-dependent file system storage
  • Included CAVE automatic virtual environment with support of unlimited nodes and projections
  • The most secure and strongest web server Seaside 3 and content managment system Pier 2 for forum pages and web-services
  • OSC support for TUIO, Kinect and WebApp controllers through Seaside
  • OMeta for Squeak and user-defined markup languages
  • Multiplatform: running on Winodws/Mac/Linux
  • Docker container (soon)

CAVE automatic virtual environment

  • Unlimited nodes and projections
  • In space CAVE editor
  • Head tracking (soon)
  • Live coding on running CAVE

Live Coding on Smalltalk

  • Create user-defined grammars, parsers, tokenisers using OMeta for Squeak
  • Open Sound Control messages support (OSC)
  • TUIO protocol support
  • Create Web apps on Seaside for controlling Virtual World from Web browser

Collaborative and User friendly

  • Forums and templates of virtual worlds are avaliable by default
  • Local and network scenarios for classroom/exhibition/research
  • Decentrallized application holds the same simulation for all participants
  • Batteries included - a lot of objects, functions, operations avaliable in Virtual World

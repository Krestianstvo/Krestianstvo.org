Krestianstvo SDK


Video of Krestiasntvo SDK v.1.0


Current version of Krestiasntvo SDK - version 3.0 (Details)

Krestianstvo SDK is an ecosystem of applications programmed on SmallTalk and JavaScript programming languages for the "3-dimensional operating system" Open Croquet / Open Qwaq / VWF, which are run on top of the Smalltalk virtual machine or JavaScript VM in the web-browser. Several running instances of the application organize a network similar to torrent (p2p) and provide distributed computation, based on replication in a single shared virtual world.

Versions

Krestianstvo SDK 3.0 (current version)

Virtual Worlds in web-browser - application based on the open source version of VWF and A-Frame components for web-browser and Web XR standarts, VR/AR/MR headsets and glasses. It is runnable actually on all avaliable platforms and devices, that support Web XR (Google Chrome / Mozilla Firefox / Safari). The main programming language is JavaScript. (Details)

Krestianstvo SDK 2.0

Virtual Worlds for a desktop - application based on the open source version of Open Qwaq/Open Croquet, which uses Smalltalk VM for Windows / Mac OS X / Linux. SDK includes a lot of applications from video/audio chat to CAVE (automatic virtual environment). The main programming language is SmallTalk. (Details)

Krestianstvo SDK 1.0

The prototype of a Virtual Worlds for a desktop - application based on the open source version of the original Open Croquet, which uses Smalltalk VM for Windows / Mac OS X / Linux. Includes the prototypes of the modules - "Smotrini", "Igrishe", "Obryad". Also there are a lot of applications inside SDK, including CCSE - collaborative curved space explorer. The main programming language is SmallTalk. (Details)

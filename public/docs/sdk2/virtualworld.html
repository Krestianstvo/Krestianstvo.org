Virtual World

On that site the virtual world's server is running!

The address for connection from Крестьянство SDK is: krestianstvo.org

To enter the World/Forum you need to get the NAME and PASSWORD. Please, send the request to the following address: register@krestianstvo.org or fill online register form at https://my.krestianstvo.org.

On all questions and difficulties, like forgotten password, problems with login,etc. write to: register@krestianstvo.org, feel free to discuss it in Krestianstvo's community pages.

Feel free to try out the running virtual world! The public logins and passwords for krestianstvo.org

Login: member
Password: member
Server: krestianstvo.org
Organization: krestianstvo

You could start your own local server. For that enter in organization field.

Default login for local server is:

Login: admin
Password: admin

If you start the server locally, you could add the members to it's local storage database. In main login screen use right mouse button to have the context menu and select: "Add new user...". You could add as new members and new groups also.

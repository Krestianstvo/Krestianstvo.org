Крестьянство SDK 1.0

Крестьянство SDK - инструментарий для создания мультимедиа приложений с поддержкой сетевых технологий на основе репликации. Может применяться при создании распределенных вычислительных мультимедиа сред; мультимедиа дисков; интернет приложений, обучающих приложений по научным дисциплинам (математика/ физика/ химия) и их прикладным областям.Может применяться для построения виртуальных лабораторий; многопользовательских обучающих сред; моделирования физических процессов.

Является SmallTalk приложением для OpenCroquet, включает Sophie XUL логику разметки пользовательских интерфейсов и пространств.

Обеспечивает выполнение следующих функций:

  • психологически-комфортная форма представления знаний
  • мультимедийный пульт учителя
  • создание среды для творческого коллективного виртуального взаимодействия 2D или 3D в реальном времени.
  • язык высокого уровня OWML (Objective World Markup Language) - язык разметки "объективного мира" в компьютерах на основе XML
  • XUL логика построения пользовательских интерфейсов приложений.
  • импорт мультимедиа содержания (2D, 3D, звук, видео) реального времени на основе библиотек OpenGL, OpenAL.
  • технологии моделирования в реальном времени (OpenGL, GLSL shaders, OpenAL)
  • поддержка Unicode шрифтов

Документация
  • На основе Open Qwaq архитектуры распределенных вычислений Open Croquet
  • Мультиплатформенное приложение: запускается на операционных системах Winodws/Mac/Linux
  • Нулевая инсталляция - запуск приложения без предварительной установки или настройки
  • Среда программирования Squeak открытая реализация языка Smalltalk
  • NoSQL CouchDB база данных для сериализации пользовательских данных
  • В комплекте CAVE - автоматическая система виртуальной реальности с поддержкой неограниченного количества проекций и сетевых узлов
  • Один из самых надежных и безопасных веб-серверов Seaside 3 с системой упралением содержанием Pier 2 для веб сервисов
  • Поддержка протоколов OSC, TUIO, Kinect и контроллеров через веб-браузер, сервер Seaside
  • OMeta для Squeak для создания пользовательских языков разметки
  • Docker контейнер для сервера (скоро)

CAVE автоматическая система виртуальной реальности

  • Неограниченное число узлов и проекций
  • Редактор CAVE в виртуальном пространстве
  • Трекинг головы (скоро)
  • Живое программирование на работающей CAVE

Живое программирование на Smalltalk

  • Программирование компонентов приложения внутри Виртуального пространства
  • Создание пользвательских грамматик, парсеров, предметных языков разметки на OMeta для Squeak
  • Протокол Open Sound Control (OSC) для обмена сообщениями между программами и цифровыми интерфейсами
  • Протокол TUIO
  • Создание веб приложений на основе Seaside для управления объектами виртуального пространства из веб-браузера

Совместная работа и доступность пользователям

  • Форумы и шаблоны виртуальных пространств доступны по умолчанию
  • Богатый функционал - множество созданных объектов, поведений, операций доступных напрямую в виртуальном пространстве
  • Локальные и сетевые сценарии созданы для приминения в классе, на выставке, в исследовательской деятельности
  • Децентраллизованное приложение обеспечивает целостность симуляции, запущенной на всех участниках виртуального пространства

Документация

  • На основе архитектуры распределенных вычислений Open Croquet
  • Среда программирования Squeak открытая реализация языка Smalltalk
  • Sophie XUL для разметки пользовательских интерфейсов
  • Нулевая инсталляция - запуск приложения без предварительной установки или настройки
  • Мультиплатформенное приложение: запускается на операционных системах Winodws/Mac/Linux
  • В комплекте CAVE - автоматическая система виртуальной реальности с поддержкой неограниченного количества проекций и сетевых узлов
  • В комплекте CCSE многопользовательский исследователь искривленных пространств
  • Один из самых надежных и безопасных веб-серверов Seaside
  • Поддержка протоколов OSC, TUIO, Kinect и контроллеров через веб-браузер, сервер Seaside
  • OMeta для Squeak для создания пользовательских языков разметки

Виртуальное обучающее пространство

  • Совместная работа в 3D Виртуальных мирах
  • Порталы
  • Фильтры
  • Интеграция с Веб-сервером
  • OSC и Midi протокол (TUIO, Wii, Kinect)
  • Squeak/Etoys для взаимодействия с виртуальным пространством
  • Совместное живое программирование
  • Библиотека объектов

Многопользовательский исследователь искривленных пространств (CCSE)

  • Совместное исследование структуры искривленных пространств
  • Визуализация искривленных пространств в реальном времени
  • Реализация алгоритма визуализации автора Jeff Weeks на языке Smalltalk для архитектуры вычислений OpenCroquet
  • Сложные симуляции и глубокое погружение
  • SnapPea совместимо - приложение по изучению многообразий
  • Богатая библиотека сгенерированных пространств: Евклидово, Гиперболическое, Сферическое

Проекты

  • Мультимедийный диск «Уистлер и Россия», Государственная Третьяковская галерея (Москва)
  • Мультимедийный диск «А.А. Иванов. Рисунок. Акварель. Библейские Эскизы», государственная Третьяковская галерея (Москва)
  • Мультитач стол для совместной работы

  • Виртуальный музей геометрий 3D

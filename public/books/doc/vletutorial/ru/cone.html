Конус

Пирамида с круглым поперечным сечением.

• Height - по умолчанию 1, диапазон 0,00 - 10,00.

• Height Segments - по умолчанию 1, диапазон 1 - 16.

• Radius - по умолчанию 1, диапазон 0,00 - 10,00.

• Radius Segments - по умолчанию 10, диапазон 1 - 16.

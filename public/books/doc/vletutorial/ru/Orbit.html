Орбита

Орбита

Для задания поведения объекту Орбита выберите из меню Создать /Поведения/Орбита. Откройте свойства объекта. Тогда откроется диалоговое окно для настройки поведения Орбита:

В этом окне можно задать координаты центра вращения объекта по орбите, скорость движения, радиус орбиты и выбрать оси координат, вокруг которых будет вращаться объект.

Упражнение 2 1. Выберите куб.

  1. Выберите из меню Создать/Поведения/Орбита

  2. Запустите плеер для тестирования поведения объекта. Куб должен вращаться вокруг оси z по заданной орбите.

  3. Меняйте радиус орбиты, скорость вращения, значения осей координат и тестируйте поведение куба при каждом новом изменении. Что при этом наблюдаете?

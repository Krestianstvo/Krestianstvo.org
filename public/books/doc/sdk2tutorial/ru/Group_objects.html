Группа объектов

Создание группового объекта.

Группировка объектов может обеспечить более удобный способ работы над объектами.

Чтобы сгруппировать несколько объектов, необходимо выполните следующие действия:

  1. Нажимая и удерживая клавишу Shift на клавиатуре, поочередно щелкаете объекты в 3-D сцене. При выборе каждого объекта, его окружает контейнер. Несколько выбранных объектов отображаются несколькими редакционными контейнерами, по одному для каждого выбранного.

  2. Щелкните правой кнопкой мыши выбранный объект и выберите Group (группу) из контекстного меню, которое появляется. Один контейнер окружает все объекты, которые группа содержит. Теперь вы можете работать над группой, щелкнув правой кнопкой мыши и выбирая элементы из контекстного меню:

  1. Установите флажок Atomic в окне Инспектора свойств (Properties). Галочка в Atomic указывает, что при нажатии любого объекта в группе выбирается вся группа целиком.

Задание

  1. Выберите из списка объектов цилиндр и конус, и разместите их в Форуме.

  2. Нажимая и удерживая клавишу Shift на клавиатуре, поочередно щелкаете конус и цилиндр в 3-D сцене левой кнопкой мыши.

  3. Щелкните правой кнопкой мыши по конусу и выберите Group (группу) из контекстного меню, которое появляется.

  4. Щелкните правой кнопкой мыши по любому из объектов и выберите из контекстного меню Properties.

  5. Установите флажок Atomic в окне Инспектора свойств (Properties) и закончите кнопками Apply и ок.

Список действий над выбранными объектами

Отменить действие Ctrl-Z

Отменяет последнее действие. Большинство действий, добавлять, удалять или изменять Форум содержание может быть отменено. Действия, которые не изменяют содержание или по своей природе невыполнимы не могут быть отменены.

Повторить действие Ctrl-Y

Восстанавливает результат последнего отмененного действия.

Вырезать Ctrl-Х

Удаляет выбор из 3-D сцены, и помещает его в буфер

Копировать Ctrl-С

Добавляет выбор в буфер без удаления Выбранного из 3-D сцены.

Вставить Ctrl-V

Разместить копию из буфера в 3-D сцену.

Удалить Delete

Удаляет текущий выбор со сцены, не помещая его в буфер.

Найти Locate sel

Изменяет вашу точку обзора, чтобы показать текущий выбор.

Задание

  1. Скопируйте группу из конуса и цилиндра в другое место в Форуме.

  2. Старую группу удалите.

  3. Отмените последнее действие.

  4. Примените команду Найти из меню Правка к выделенной группе объектов.
